# CRM Modeling — RAKBANK CRM 360

Banking CRM 360 logical data model derived from the field requirements workbook.

## Artefacts

| File | Description |
|------|-------------|
| [`data-model.md`](./data-model.md) | Complete entity catalogue — all entities, attributes (with data types, constraints, descriptions), relationships, and design decisions |
| [`data-model.mmd`](./data-model.mmd) | Mermaid ER diagram — paste into [mermaid.live](https://mermaid.live) or any Mermaid-compatible renderer |
